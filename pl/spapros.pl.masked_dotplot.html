spapros.pl.masked_dotplotÔÉÅ

spapros.pl.masked_dotplot(adata, selector, ct_key='celltype', imp_threshold=0.05, celltypes=None, n_genes=None, comb_markers_only=False, markers_only=False, cmap='Reds', comb_marker_color='darkblue', marker_color='blue', non_adata_celltypes_color='grey', save=False)ÔÉÅ

Create dotplot with additional annotation masks.

Parameters

  • adata (AnnData) ‚Äì AnnData with cell type annotations in adata.obs[ct_key].

  • selector ‚Äì ProbesetSelector object with selected selector.probeset.

  • ct_key (str) ‚Äì Column of adata.obs with cell type annotation.

  • imp_threshold (float) ‚Äì Annotate genes as ‚ÄúSpapros marker‚Äù only for those genes with importance > imp_threshold.

  • celltypes (Optional[List[str]]) ‚Äì Optional subset of celltypes (rows of dotplot).

  • n_genes (Optional[int]) ‚Äì Optionally plot top n_genes genes.

  • comb_markers_only (bool) ‚Äì Whether to plot only genes that are ‚ÄúSpapros markers‚Äù for the plotted cell types. (can be combined with markers_only, in that case markers that are not Spapros markers are also shown)

  • markers_only (bool) ‚Äì Whether to plot only genes that are markers for the plotted cell types. (can be combined with comb_markers_only, in that case Spapros markers that are not markers are also shown)

  • cmap (str) ‚Äì Colormap of mean expressions.

  • comb_marker_color (str) ‚Äì Color for ‚ÄúSpapros markers‚Äù.

  • marker_color (str) ‚Äì Color for marker genes.

  • non_adata_celltypes_color (str) ‚Äì Color for celltypes that don‚Äôt occur in the data set.

  • save (Union[bool, str]) ‚Äì If True or a str, save the figure.

Example

(Takes a few minutes to calculate)

import spapros as sp
adata = sp.ut.get_processed_pbmc_data()
selector = sp.se.ProbesetSelector(adata, "celltype", n=30, verbosity=0)
selector.select_probeset()

sp.pl.masked_dotplot(adata,selector)
